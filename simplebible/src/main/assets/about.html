Let me begin by saying, “I am thankful and grateful to you” for using this application. It means the application is helpful at atleast one person and I hope it serves a more meaningful purpose than a simple reference to the Holy Bible.

Simple Bible intends to be easy to use & avoid unnecessary features.

These were the two most common requests the elders I knew wished in the Bible applications they used on their phones. So began my task and this is my second attempt to meet their needs.

For suggestions, feedback or anything else, use the "Email Developer" option on the Settings screen.

This is an Open Source application, you can find the source code hosted on Github here.

Credit and Recognition
